# Open-Blog

## Project Setup

### Backend

1. Navigate to the `backend` directory:
    ```sh
    cd backend
    ```

2. Build the backend:
    ```sh
    ./mvnw clean install
    ```

3. Run the backend:
    ```sh
    ./mvnw spring-boot:run
    ```

### Frontend

1. Navigate to the `frontend` directory:
    ```sh
    cd src/main/resources/static/frontend
    ```

2. Install the dependencies:
    ```sh
    npm install
    ```

3. Start the frontend development server:
    ```sh
    npm start
    ```
